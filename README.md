## Setup
```bash
npm install

```
## 目录结构
<pre>
|	assets.json		# 需要打包编译的目录配置文件

|	README.md		# 项目说明
|_src
	|_assets		# 存放公共css、js文件	例如（jquery.js、reset.css）
	|	css
	|	js
	|
	|_es6			# 编写es6语法的js文件
	|
	|_images		# 图片存放目录
	|
	|_scss			# sass文件
	|
	|_views			# 页面
	|
	|	index.html	# 主页面
</pre>
